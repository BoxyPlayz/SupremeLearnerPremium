# Axioms and Postulates

They're true statements which are just true because [Wynaut](https://www.pokemon.com/us/pokedex/wynaut), so I guess here's a list.

## Segment Addition Postulate

If B is between A and C, then AB+BC=AC  
If AB+BC=AC, then B is between A and C

## Angle Classification

If an angle is 90°, then it is a right angle  
If an angle is 180°, then it is a straight angle  
If an angle is less than 90°, then it is an acute angle  
If an angle is more than 90°, but less than 180°, then is is a ~~Big Back~~ "obtuse angle"

## Angle Addition Postulate

If C is in the interior of ∠BAD, then m∠BAC=m∠CAD+m∠BAD

## 2 Point Postulate

If there are 2 points, then there is EXACTLY one line

## Line-Point Postulate

If there is a line, then the line has AT LEAST 2 points

## Point-Plane Postulate

If there are 3 noncolinear points, then there is EXACTLY 1 plane

## The Bee Movie Postulate

If there are known laws of aviation, no bee can fly

## Plane-Point Postulate

If there is a plane, it has 3 noncolinear points

## Line-Plane Postulate

If 2 points lie on a plane, then the line containing them is also on the plane

## Parallel Postulate

If there is a line and a point that's not on the line, then there is EXACTLY one line that goes through the point and is parallel to the given line

## Purpendicular Postulate

If there is a line and one point not on the line, then there is EXACTLY one line that goes through the point purpendicular to the given line

## Triangle Classification

An equilangular triangle has all angles be 60°. It is also always equilateral.  
A right triangle has exactly one angle that is 90°  
An obtuse triangle has one obtuse angle  
An acute triangle ONLY has acute angles  
An isosceles triangle has at least 2 congruent sides  
An equilateral triangle has all sides be congruent. It is also always equilangular.  
A scalene triangle has no congruent sides  
For more information, see [Angle Classification](#angle-classification)

## Interior and Exterior Angle Triangle Sums

The sum of the interior angles of a triangle is ALWAYS 180°  
The sum of the exterior angles of a triangle is ALWAYS 360°  
1 exterior angle is the sum of 2 non-adjacent angles  
The 2 acute angles of a right triangle are complimentary

## Angle Angle Similarity

If 2 angles in one triangle are congruent to 2 angles in a second triangle, then the triangles are similar

## Side Side Side Similarity

If the sides in 1 triangle are proportionate to the sides in the second triangle, then the triangles are similar

## Side Angle Side Similarity

If 2 sides in 1 triangle are proportionate to 2 sides in a second triangle and the included angle in 1 triangle is congruent to included angle in second triangle, then the triangles are similar
